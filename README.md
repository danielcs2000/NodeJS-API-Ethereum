# NodeJS-API-Ethereum

## Prerequisites 

- Node.js
- Truffle 
- Ganache
- Web3.js
- Postman

## Setup

Install truffle and ganache with the following commands:

```bash
npm install -g truffle
```

```bash
npm install -g ganache
```

Install dependencies for the project

```bash
npm install
```

## Start API

#### Start Ethereum blockchain

```bash
ganache-cli
```

#### Compile and Migrate Smart Contract

```bash
truffle compile
```

```bash
truffle migrate
```

#### Run express server

```bash
node app.js
```
